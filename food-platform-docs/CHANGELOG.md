## Versioning

This project follows [Semantic Versioning](https://semver.org/):
- MAJOR: Breaking changes
- MINOR: New features (backward compatible)
- PATCH: Bug fixes (backward compatible)

Example: `1.2.3`
- 1 = MAJOR (breaking changes)
- 2 = MINOR (new features)
- 3 = PATCH (bug fixes)
